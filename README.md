## 🎯 Sobre o Projeto

Este é um sistema personalizado para geração de Ordens de Serviço, desenvolvido inicialmente para terminal em Kotlin e posteriormente adaptado para uma interface web com auxílio de IA. O objetivo? Simplificar e agilizar o registro de atendimentos técnicos no dia a dia.

### A História por trás do código

Atualmente, trabalho como suporte técnico e, no cotidiano, precisamos gerar Ordens de Serviço para registrar os atendimentos — tudo feito no bloco de notas. Para agilizar esse processo, desenvolvi um sisteminha em Kotlin que roda direto no terminal. Simples, funcional e feito sob medida para as necessidades da minha rotina.

Mas aí pensei: poderia ser melhor, mais prático do que ter que abrir o IntelliJ todo dia. Foi quando decidi dar o próximo passo: criar uma interface web.

## 🤖 O Papel da IA

Este projeto é um exemplo interessante de como a IA pode acelerar o desenvolvimento de soluções práticas:

1. **Comecei com o código Kotlin** - Funcional, mas preso ao terminal
2. **Usei IA para gerar o front-end** - Descrevi minha necessidade e, em instantes, recebi uma interface limpa e funcional
3. **Integrei as partes** - O resultado é um sistema coeso que mantém a lógica original com uma apresentação moderna

## ✨ Funcionalidades

### Tipos de OS disponíveis:
- 🔧 **Visita Técnica** - Avaliação técnica no local
- 🛠️ **Manutenção Corretiva** - Correção de problemas
- 📦 **Instalação** - Instalação de equipamento
- 📡 **Instalação de Repetidores** - Repetidores de sinal
- 🔄 **Reativação** - Reativação de serviço
- 🔁 **Troca de Equipamento** - Substituição de equipamento
- 📤 **Retirada de Equipamento** - Retirada do local
- 📍 **Avaliação de Realocação** - Verificar possibilidade de realocação
- 🚚 **Realocação de Equipamento** - Mudança de local do equipamento

### Para cada OS, é possível registrar:
- ✅ Data e horário do atendimento
- ✅ Presença do cliente
- ✅ Local de instalação/sugestão
- ✅ Porta e sinal do equipamento
- ✅ Questões de infraestrutura
- ✅ Integridade dos equipamentos
- ✅ Observações
- ✅ Assinaturas digitalizadas (técnico e cliente)
- ✅ CPF do responsável (opcional)
